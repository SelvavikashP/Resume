📊 Student Marks Management System (Excel)
📌 Description

This project is an Excel-based Student Marks Management System designed to manage and analyze student academic performance efficiently. It automates calculations and provides clear insights using built-in Excel features.

✨ Features

Automatic calculation of Total and Average marks

Grade assignment using conditional formulas

Easy-to-use and well-structured spreadsheet

Suitable for academic and small-scale educational use

🛠 Tools & Concepts Used

Microsoft Excel

Excel Formulas (SUM, AVERAGE, IF)

Basic data organization and analysis
